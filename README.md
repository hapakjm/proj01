# HTML & CSS Tutorial Project

A simple **HTML/CSS project** demonstrating basic web development concepts.

## Description

This project is built following the YouTube tutorial by **@RozanChitrakar** titled **"How To Create A Website using HTML & CSS | Step-by-Step Tutorial"**.

[Tutorial Link](https://www.youtube.com/watch?v=CQZxeoQeo5c&list=WL&index=14)

## Features

- Responsive design
- Clean and organized code structure
- Basic HTML and CSS implementation

## Technologies Used

- HTML5
- CSS3

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/hapakjm/proj01.git
   ```
2. Navigate to the project directory:
   ```bash
   cd proj01
   ```
3. Open `index.html` in your web browser to view the project.

## Usage

Simply open the `index.html` file in any modern web browser to see the website in action.
